All about Display and Positioning!

Position Explanation
Static The default positioning, this positions the element to the left
Relative This must be accompanied by top, bottom, left or right values. It moves the element away.
Absolute The element will be positioned relative to the nearest ancestor, if there is no ancestor then it will be positioned relative to the body and WILL move will scrolling. This allows the element to overlap others
Fixed The element will always stay in place relative to the viewport, even when scrolled. It can be modified using top, bottom, left or right.
Sticky The element is positioned normally until the user scrolls past, then it will stick to the top of the viewport
